# YumekaOS

### Keybinds

| Keybind    | Description         |
|------------|---------------------|
| leader + e | File Explorer       |
| leader + e | Telescope           |
| leader + e | Find Grep           |
| leader + e | Quit Neovim         |

### Harpoon

| Keybind    | Description                |
|------------|----------------------------|
| leader + a | Add current buffer to list |
| ctrl + e   | Open buffer list           |
| ctrl + n   | Go backwards on list       |
| ctrl + p   | Go foward on list          |
