# YumekaOS

### General

| Keybind         | Description         |
|-----------------|---------------------|
| `leader` + `qq` | Quit Neovim         |

### Telescope

| Keybind         | Description         |
|-----------------|---------------------|
| `leader` + `ff` | Telescope           |
| `leader` + `fg` | Find Grep           |

### Harpoon

| Keybind        | Description                |
|----------------|----------------------------|
| `leader` + `a` | Add current buffer to list |
| `ctrl` + `e`   | Open buffer list           |
| `ctrl` + `n`   | Go backwards on list       |
| `ctrl` + `p`   | Go foward on list          |

### Mini Files

| Keybind         | Description         |
|-----------------|---------------------|
| `leader` + `e`  | File Explorer       |
